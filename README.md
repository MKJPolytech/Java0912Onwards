단순 문법 암기가 아닌, 코드 구조와 실행 흐름을 이해하는 데 중점을 두었습니다.

## 4. 프로젝트를 통해 배운 점

객체지향 프로그래밍의 기본 개념을 학습하며

## 5. DB 연동 도서관리 시스템 프로젝트

Java와 데이터베이스 연동을 학습하기 위해  
도서 정보를 관리하는 **콘솔 기반 도서관리 시스템**을 구현하였습니다.

### 프로젝트 개요
본 프로젝트는 단순한 Java 문법 실습을 넘어,  
**DB와 연동된 CRUD 기능을 직접 구현**하는 것을 목표로 진행되었습니다.  
사용자는 콘솔 환경에서 도서 정보를 등록, 조회, 수정, 삭제할 수 있습니다.

### 주요 기능
- 도서 정보 등록 (도서명, 저자, 출판사 등)
- 도서 목록 조회
- 도서 정보 수정
- 도서 정보 삭제
- 데이터베이스 연동을 통한 영속성 관리

### 기술적 특징
- Java 기반 콘솔 애플리케이션
- JDBC를 활용한 데이터베이스 연동
- SQL을 이용한 CRUD 쿼리 작성
- 애플리케이션 로직과 DB 처리 로직 분리

### 프로젝트를 통해 배운 점
이 프로젝트를 통해  
Java 애플리케이션에서 데이터베이스를 연동하는 전체 흐름을 이해하였으며,  
단순 로직 구현을 넘어 **데이터 설계와 관리의 중요성**을 체감하였습니다.  
이 경험은 이후 Spring Boot와 JPA 기반 웹 애플리케이션을 학습하는 데  
기초적인 토대가 되었습니다.
