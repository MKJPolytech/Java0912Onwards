# Java0912Onwards – Java 기초 및 객체지향 학습 프로젝트

Java 프로그래밍 언어의 기초 문법과  
객체지향 개념을 단계적으로 학습하며 작성한 실습 코드 모음입니다.

---

## 1. 프로젝트 개요

본 저장소는 Java 언어의 기본 문법부터  
클래스와 객체 개념을 이해하기까지의 학습 과정을 정리한 프로젝트입니다.  
단순 문법 암기가 아닌, 코드 구조와 실행 흐름을 이해하는 데 중점을 두었습니다.

---

## 2. 주요 학습 내용

- 변수, 조건문, 반복문 등 Java 기본 문법
- 클래스와 객체 개념 이해
- 메서드 정의 및 호출 구조
- 간단한 로직을 객체 단위로 구현하는 연습

---

## 3. 사용 기술

- Java

---

## 4. 프로젝트를 통해 배운 점

객체지향 프로그래밍의 기본 개념을 학습하며  
코드를 기능 단위로 분리하고 책임을 나누는 사고방식을 익혔습니다.  
이 경험은 이후 Spring Boot 기반 백엔드 개발과  
전체 시스템 구조를 이해하는 데 중요한 기반이 되었습니다.
